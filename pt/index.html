FAQ

Perguntas frequentes.

O Z-Move precisa de agentes instalados nas minhas VMs?
Não. O Z-Move é agentless por design. Zero software é instalado nas suas VMs de origem ou destino em qualquer momento. Os seus workloads nunca são tocados — é utilizada apenas a camada de API cloud.
O Z-Move consegue migrar SAP HANA ou bases de dados enterprise grandes?
Sim. O Z-Move migrou SAP HANA em produção com 7 discos em 253 minutos, e clusters SQL Server com 3 discos por instância em menos de 100 minutos. Workloads enterprise são um caso de uso central.
O que acontece à infraestrutura de migração após a conclusão?
Todos os workers efémeros são automaticamente destruídos após cada job. Sem recursos persistentes, sem custos residuais e sem superfície de ataque. Cada job começa e termina limpo.
Como é calculado o preço?
O Z-Move cobra $0.05 por GB de dados transferidos. Sem taxas de configuração, sem subscrições, sem custos idle. Paga apenas pelo que realmente migra. Descontos de volume enterprise disponíveis a partir de 10 TB.
